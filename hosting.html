Hosting & Next Steps

Hosting Your Website

This website is built with HTML, CSS, and JavaScript, making it a "static site." This is great because static sites are fast, secure, and easy to host. Here are some recommended hosting platforms:

  • Netlify: Excellent for beginners. Offers a generous free tier and simple drag-and-drop deployment.
  • Vercel: Another fantastic option with a great free tier, known for its speed and developer-friendly features.
  • GitHub Pages: A free hosting service directly from your GitHub repository. Perfect if you're already using GitHub.

To deploy, you typically just need to sign up for one of these services, connect your project folder, and they handle the rest!

Implementing Email Notifications

The contact form currently saves messages to a secure database (Firebase Firestore). To receive an email notification for each new message, you need to add a piece of server-side logic. The standard way to do this with Firebase is using Cloud Functions.

Here's the basic process:

  1. Set up the Firebase CLI (Command Line Interface) on your computer.
  2. Create a new Cloud Function that is "triggered" whenever a new document is added to your 'messages' collection in Firestore.
  3. Inside this function, write code to use a service like SendGrid or Mailgun to send an email to your desired address. The email can contain the data from the form (name, email, message).
  4. Deploy the function to your Firebase project.

This creates a secure and scalable way to get instant notifications without exposing your email credentials on the client-side website.
